### Tehnology
* Java
* RxJava3
* Room database
* Retrofit 
* Glide
* MVVM

### Project description

The app will allow you to watch information about the highest-value movies and add them to your favorites

### Functional 

* Add movies to favorites
* View trailers
* Movie information (using the Kinopoisk API)
